# SMS Varanasi - Online Fee Deposit Portal (Clone)

This is a pixel-perfect clone of the SMS Varanasi online payment portal.

## Files Included
- `index.html` - Main HTML structure
- `styles.css` - Complete styling (matches original design)
- `script.js` - Interactive functionality
- `logo.png` - SMS Varanasi logo (SVG placeholder)
- `banner.png` - Online Payment banner (SVG placeholder)

## Features
✓ Exact layout matching reference images
✓ Desktop-only layout (1024px fixed width)
✓ Works on mobile devices (shows desktop version with horizontal scroll)
✓ All 6 fee deposit buttons (clickable, ready for navigation links)
✓ Complete content sections:
  - Important Note
  - Disclaimer
  - Terms and Conditions
  - Privacy Policy
  - Refund/Cancellation Policy
  - List of Services
✓ Bottom button grid
✓ Footer with credits

## How to View
1. Open `index.html` in any web browser
2. The layout is fixed at 1024px width
3. On mobile, you can scroll horizontally to see full content

## Button Configuration
All buttons are clickable but navigation is not yet configured.
Update the JavaScript in `script.js` to add your navigation URLs.

## Notes
- Logo and banner are SVG placeholders
- Replace `logo.png` and `banner.png` with actual images if available
- All styling matches the original SMS Varanasi portal
